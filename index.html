Code Spotlight

Asteroids Clone

During our third week of boot-camp we recreated the classic game Asteroids, using Pygame as our graphics engine. It was a team project involving myself and the rest of the helium team. My responsibilites were the main game engine loop, the start-up screen, and some helper classes with the explosion animation class, and the bullet class. The other guys ran the main ship, and asteroids class, and another handled all the sprite blitting, and collision detection.

Time Tracker Rev 1.0

During our second week of boot-camp we were able to choose a person project to work on. I wanted to learn about pythons Graphic User Interface libraries. This Time-Tracker widget is designed to sit on your desktop and with a simple 3 button interface, it will track the time you spend on a particular project. This version saves it directly to a tab-delimtied text file that you can import into Excel or any other spread-sheet. You can find Revesion 2 the repository as well, but it is still in progress (because the boot camp keeps us very busy). Revision 2 will incorporate MySQL integration with a remote server. Revision 3 (still in concept) will add a managers screen, to review and track time by project/budget/resources.
